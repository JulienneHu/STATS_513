(1) Let $$A = \begin{bmatrix}1 \ 0\ 0 \\ 0\ 1\ 0\end{bmatrix}$$ $$B = \begin{bmatrix}0 \ 1\\0 \ 3 \\ 1 \ 2\end{bmatrix}$$

(a) Please calculate AB.¶
(b) Please show AB is singular. (Hints: You can show the columnsof AB are linear dependent or calculate the determinant to prove it)¶

$A B = \begin{bmatrix}1 \ 0\ 0 \\ 0\ 1\ 0\end{bmatrix} \begin{bmatrix}0 \ 1\\0 \ 3 \\ 1 \ 2\end{bmatrix} = \begin{bmatrix}0 \ 1 \\ 0 \ 3 \end{bmatrix}$

$\because A B = \begin{bmatrix}0 \ 1 \\ 0 \ 3 \end{bmatrix}\\$ $\therefore 3\times(0,1)-(0,3)=0$ Therefore, $AB$ are linear dependent

Bonus: Let x1 = (1, 0, 0)>, x2 = (0, 1, 0)> and x3 = (0, 0, 1)>. Please show that x1, x2 and x3 are linear independent. (Hints: Please use proof by contradiction. Firstly, assume there exists a1, a2 and a3, where at least one of a1, a2 and a3 is non-zero, such that a1x1 + a2x2 + a3x3 = (0, 0, 0)>. Secondly, show that a1x1 + a2x2 + a3x3 = (0, 0, 0)> holds only if a1 = a2 = a3 = 0.)¶

Given that $x_1=(1,0,0)^T, x_2 = (0,1,0)^T,x_3 = (0,0,1)^T$,\ suppose $\exist a_1,a_2,$ and $a_3$ where at least one of $a_1,a_2,$ and $a_3$ is non-zero such that $a_1x_1+a_2x_2+a_3x_3=(0,0,0)^T$. Then we have:\ $(a_1,0,0)^T+(0,a_2,0)^T+(0,0,a_3)^T=(a_1,a_2,a_3)^T=(0,0,0^T)$.\ Therefore, the equation holds if and only if $a_1=a_2=a_3=0$.\ Hence, $x_1,x_2,$ and $x_3$ are linear independent.

The sample correlation between the residuals and the fitted value is -1.14138128387761e-16.\ In definition, regression will give a linear trend based on predicators and residuals are the randomness that is leftover from a regression model from fitting a regression model. The correlation between the explanatory variable(s) and the residuals is/are zero because theres no linear trend left - its been removed by the regression.\ According to vector space interception, $min_{\beta}(y-X\beta)^T(y-X\beta)$ minimizes the Euclidean distance between $y$ and the linear space spanned by the columns of X. Here the fitted data is $X\beta$, so the residuals are perpendicular to the fitted data. Hence, the correlation between the residuals and the fitted value is almost 0.
